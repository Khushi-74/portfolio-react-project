# React DevOps Portfolio 🚀

A modern React.js portfolio website built and deployed using CI/CD practices with Azure DevOps.

This project demonstrates:
- React application development
- Azure DevOps CI pipeline
- Artifact publishing
- GitHub integration
- Modern DevOps workflow

---

# Tech Stack

- React.js
- HTML5
- CSS3
- JavaScript
- Azure DevOps
- GitHub
- Node.js

---

# Features

- Responsive portfolio UI
- Skills section
- Experience timeline
- Projects showcase
- Contact section
- CI pipeline integration using Azure DevOps
